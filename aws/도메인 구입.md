# 아마존 AWS Route 53에서 도메인을 구입해보자
## [전체 AWS 문서](index.md)
작년에 sangun.link 라는 도메인을 구입했었지만, 개명한 관계로 새 도메인을 구입하려고 한다.  
구입하는 김에 과정을 기록하기로 했다.

1. AWS 콘솔에 접속한다.
    ![캡처](/aws/사진/001.png)

2. Route 53을 찾아 들어간다.
    ![캡처](/aws/사진/002.png)

3. 도메인을 클릭한다.
    ![캡처](/aws/사진/003.png)

4. 도메인 등록을 클릭한다.
    ![캡처](/aws/사진/004.png)
    
5. 구입하고싶은 도메인을 장바구니에 담고 계속 진행한다.
    ![캡처](/aws/사진/005.png)

6. 연락처를 작성한다.

7. 결제한다.
